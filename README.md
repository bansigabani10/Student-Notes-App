Flutter Notes App
A simple and efficient Flutter application that uses local storage to manage notes. This app allows users to add, view, update, and delete notes using Hive database.

Features
•	➕ Add Notes
•	📖 View Notes
•	✏️ Update Notes
•	❌ Delete Notes
•	💾 Local Storage using Hive

Technologies Used
•	Flutter
•	Dart
•	Hive Database
•	Android Studio / VS Code

Project Structure
lib/
 ┣ main.dart

Getting Started
1. Clone the repository
git clone https://github.com/bansigabani10/Student-Notes-App.git
2. Navigate to project folder
cd Student-Notes-App
3. Install dependencies
flutter pub get
4.Run the app
flutter run

App Workflow
1.	User opens the app
2.	User adds a note
3.	Note is stored in Hive database
4.	Notes are displayed on screen
5.	User can update or delete notes

Output
The application successfully performs CRUD operations using local storage and provides a simple UI for managing notes.
